# Лабораторная работа №2: Онлайн-Библиотека и Автотестирование
**Студенты:** Тихонов и Лоськов

## Описание проекта
Проект представляет собой веб-приложение "Онлайн-Библиотека" с возможностями:
* Просмотр списка доступных книг.
* Поиск книг по названию, автору или жанру.
* Добавление новых книг через форму.
* Удаление существующих записей.
* Сохранение данных в `localStorage` браузера.

## Автоматизированное тестирование
Для проекта написаны автотесты на Python с использованием **Selenium**.
Сценарии тестов включают:
1. Проверку корректного отображения начального списка книг.
2. Тестирование функции поиска.
3. Проверку добавления новой книги.
4. Проверку удаления книги.
5. Автоматическое создание скриншотов состояния страницы до и после действий.

## Как запустить тесты
1. Установите необходимые библиотеки:
   `pip install selenium`
2. Убедитесь, что у вас установлен Chrome и соответствующий ChromeDriver.
3. Запустите файл тестов:
   `python test_library.py`
