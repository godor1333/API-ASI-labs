# Генератор мемов

Веб-приложение для создания мемов с возможностью загрузки изображений, добавления текста поверх фото и скачивания готовых мемов.

## Функциональность

- ✅ Регистрация и авторизация пользователей (JWT)
- ✅ Загрузка изображений
- ✅ Добавление текста поверх изображения
- ✅ Скачивание созданных мемов
- ✅ PostgreSQL база данных
- ✅ Docker и docker-compose для всех сервисов

## Технологии

- **Backend**: FastAPI, SQLAlchemy, PostgreSQL
- **Frontend**: HTML, CSS, JavaScript
- **Аутентификация**: JWT (JSON Web Tokens)
- **База данных**: PostgreSQL
- **Контейнеризация**: Docker, Docker Compose

### Требования

- Docker
- Docker Compose

### Инструкция по запуску

1. Перейдите в директорию проекта:
```bash
cd "gen mem"
```

2. Запустите все сервисы через docker-compose:
```bash
docker-compose up --build
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost
```

### Аутентификация

- `POST /api/auth/register` - Регистрация нового пользователя
- `POST /api/auth/login` - Авторизация
- `GET /api/auth/me` - Получение информации о текущем пользователе

### Мемы

- `POST /api/memes/upload` - Загрузка изображения
- `POST /api/memes/create/{meme_id}` - Создание мема с текстом
- `GET /api/memes/download/{meme_id}` - Скачивание мема
- `GET /api/memes/list` - Список всех мемов пользователя

## Использование

1. Зарегистрируйтесь или войдите в систему
2. Загрузите изображение через форму загрузки
3. Введите текст(ы), который хотите добавить на изображение
4. Нажмите "Создать мем"
5. Скачайте готовый мем из списка ваших мемов

## Остановка проекта

```bash
docker-compose down
```

Для полной очистки (включая данные БД):
```bash
docker-compose down -v
```

## Тестирование

### Запуск тестов Selenium

1. Убедитесь, что приложение запущено:
   ```bash
   docker-compose up
   ```

2. Установите зависимости для тестов:
   ```bash
   cd tests
   pip install -r requirements.txt
   ```

3. Запустите тесты:
   ```bash
   pytest test_meme_generator.py -v
   ```

Подробнее о тестах см. [tests/README.md](tests/README.md)
