# Лабораторная работа №5: Георадар социальных сигналов — Арзамас
**Студент:** Тихонов

## Описание проекта
«Георадар» — это система мониторинга инфопотока, предназначенная для поиска локальных всплесков социальной активности в городе Арзамас. Система выявляет «социальные землетрясения» — микровсплески обсуждений, которые еще не переросли в глобальные тренды, и привязывает их к конкретным локациям (улицам и достопримечательностям).

### Основной функционал:
* **Парсинг данных:** Сбор 10 000 новостей из пабликов Арзамаса (VK, TG). Учитываются все записи, включая рекламные посты.
* **Геопривязка:** Автоматическое извлечение названий улиц и достопримечательностей Арзамаса для отрисовки тепловых точек на карте.
* **Аналитика:** Расчет «температуры» инфопотока, выявление эпицентров и скорости распространения тем.
* **Личный кабинет:** Визуализация карты активности, жизненный цикл инфоповода и прогнозы затухания.

## Используемые ML-модели
Проект использует современные предобученные модели из репозитория Hugging Face:
1. **Кластеризация постов:** `sentence-transformers/paraphrase-multilingual-mpnet-base-v2` — превращение постов в эмбеддинги и формирование тематических облаков.
2. **Температура инфопотока:** `distilbert-base-uncased` — анализ интенсивности и направленности волн интереса.
3. **Детекция аномалий:** `defog-llm-anomaly-detection` — выделение микровсплесков на фоне обычного новостного шума.

## Технический стек
* **Язык:** Python 3.x
* **Брокер сообщений:** Apache Kafka (используется `core/kafka_utils.py` для передачи сигналов в реальном времени).
* **База данных:** PostgreSQL/SQLite (конфигурация в `database/db_config.py`).
* **Парсер:** Модуль `parser/vk_parser.py` для сбора данных из социальных сетей.
* **Контейнеризация:** Docker Compose для развертывания инфраструктуры.


## Структура проекта
* `core/` — утилиты для работы с геоданными и Kafka.
* `models/` — логика загрузки ML-моделей (`model_loader.py`) и процессоры обработки сигналов.
* `parser/` — скрипты для сбора 10к новостей из источников.
* `data/streets.txt` — справочник гео-объектов Арзамаса для сопоставления.
* `dashboard.py` — визуализация тепловой карты и графиков затухания.
