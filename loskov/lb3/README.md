# Лабораторная работа №4: Онлайн MMORPG "Три в ряд"
**Студент:** Тихонов

## Описание проекта
Проект реализует механику сетевой игры "Три в ряд" с ролевой составляющей. Основной фокус сделан на выносе игровой логики на сервер и использовании брокера сообщений для синхронизации данных.

### Игровые особенности:
* **PvP/PvE матчи:** Битва до 0 HP. Базовое здоровье — 100 единиц.
* **Экипировка:** 5 уровней качества предметов, влияющих на HP персонажа.
* **Рейтинг:** Расчет Эло для определения позиции игрока в глобальном топе.
