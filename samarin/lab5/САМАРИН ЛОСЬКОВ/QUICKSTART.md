# Быстрый запуск проекта

## Способ 1: Через bat-файл (рекомендуется)

Дважды кликните на файл `start_web.bat` в проводнике Windows.

Или запустите из командной строки:
```cmd
start_web.bat
```

## Способ 2: Через Python напрямую

Откройте командную строку или PowerShell в папке проекта и выполните:

```bash
python run_web.py
```

Или:

```bash
python test_run.py
```

## Способ 3: Через главное меню

Запустите `start.bat` для выбора режима работы:
- Веб-интерфейс
- Генерация и оценка видео
- Регрессионный бенчмарк

## После запуска

1. Откройте браузер
2. Перейдите на адрес: **http://localhost:5000**
3. Вы увидите веб-интерфейс с аналитикой

## Установка зависимостей (если нужно)

Если при запуске возникают ошибки импорта, установите зависимости:

```bash
pip install -r requirements.txt
```

## Примечания

- Для генерации видео требуется запущенный ComfyUI (см. INSTALL.md)
- Веб-интерфейс работает и без ComfyUI, но будет показывать только результаты оценки существующих видео
- Если порт 5000 занят, измените порт в файле `run_web.py` или `frontend/app.py`
