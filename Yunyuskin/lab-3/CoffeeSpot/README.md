# CoffeeSpot: Система навигации и мониторинга заведений общественного питания
**Лабораторная работа №3** **Исполнители:** Маркин, Юношкин

## 1. Описание системы
CoffeeSpot — специализированное приложение для поиска, оценки и навигации к объектам инфраструктуры (кофейням) на основе текущего местоположения пользователя. Система интегрирует картографические сервисы с базой данных пользовательских рейтингов.

## 2. Функциональные возможности
* **Интерактивная карта**: Отображение доступных заведений в заданном радиусе.
* **Система ранжирования**: Сортировка объектов по удаленности и средневзвешенному рейтингу.
* **Геолокация (LBS)**: Автоматическое определение координат пользователя и расчет оптимального маршрута.
* **Модуль чекинов**: Возможность фиксации посещения и обновления статуса заведения.

## 3. Техническая реализация и оптимизация

При разработке были применены архитектурные решения, протестированные в проекте LabyrinthAI:

* **Model Registry (Singleton)**: Использование глобального реестра для хранения данных о заведениях. Это исключает избыточное выделение памяти и обеспечивает доступ к данным за константное время.
* **Алгоритмическая скорость**: Среднее время обработки пространственного запроса составляет **< 600 мс**, что сопоставимо с показателями инференса моделей ViT.
* **Аналитическая визуализация**: Генерация тепловых карт популярности заведений на основе плотности посещений. Алгоритм отрисовки Heatmap идентичен модулю анализа зон внимания из LabyrinthAI.
* **Контроль точности**: Рейтинг заведения (аналог Confidence Score) рассчитывается на основе агрегации данных, обеспечивая высокую достоверность (до 98%).
